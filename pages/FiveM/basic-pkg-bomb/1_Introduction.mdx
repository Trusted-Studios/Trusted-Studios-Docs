# FiveM Farming Plus Script

"FiveM Basic Package Bomb" is a simple yet powerful system designed for explosive immersion. Utilizing the ox_target library, this script ensures the best performance and synchronized gameplay. Deploy packet bombs strategically and create a dynamic multiplayer experience. Elevate your FiveM server with thrilling chaos and excitement.

## Installation

<Steps>
    ### Download
    Download the script from [GitHub](https://github.com/GermanWarthog/esx_GMW-PackageBomb/releases/latest)

    ### Installation
    Drag and Drop the resource into your server-files and go to the configuration file located at `/conig.lua`.

    ### Checking Dependencies
    To run the script you'll need to have [es_extended](https://github.com/esx-framework/esx_core/releases/latest) & [ox_target](https://github.com/overextended/ox_target/releases/latest) installed on your server.

    ### Configuration
    Please make sure to configure the script to your need to ensure the best experience. More information can be found [here](./2_Config).
</Steps>
