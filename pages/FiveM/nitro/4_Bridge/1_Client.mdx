# Client Functions

## HaveItem

```lua
frameworkName:HaveItem(item, count)
```

- item: `string` (item name)
- count: `number` (needed amount)
- returns: `boolean`

### Example 

```lua copy 
function ESX:HaveItem(item, count)
    local HaveItem = Async.Await(function(promise)
        ESX.Shared.TriggerServerCallback('Trusted:Nitro:HaveItem', function(haveItem) 
            promise:resolve(haveItem)
        end, item, count)
    end)

    return HaveItem
end
```

## UseItem

```lua
frameworkName:UseItem(item, count)  
```

- item: `string` (item name)
- count: `number` (amount)

### Example

```lua copy 
function ESX:UseItem(item, count)
    TriggerServerEvent('Trusted:Nitro:ESX:UseItem', item, count)
end
```
