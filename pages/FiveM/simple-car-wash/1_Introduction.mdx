# FiveM FiveM Simple Car Wash

The car wash script provides a sponge item, wich enables players to wash their cars.

## Installation

<Steps>
    ### Download
    Download the script from [GitHub](https://github.com/GermanWarthog/ts_esx-CarWash/releases/latest)

    ### Installation
    Drag and Drop the resource into your server-files and go to the configuration file located at `/conig.lua`.

    ### Configuration
    Please make sure to configure the script to your need to ensure the best experience. More information can be found [here](./2_Config).
</Steps>
