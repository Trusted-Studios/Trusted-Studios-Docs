# FiveM Farming Plus Script

The FiveM Basic Job Garage is a straightforward Lua script designed for FiveM that empowers players to efficiently manage a dedicated garage for their job. With this script, players can easily park and unpark their vehicles, ensuring convenient access and organization.

## Installation

<Steps>
    ### Download
    Download the script from [GitHub](https://github.com/GermanWarthog/esx_GMW-JobGarage/releases/latest)

    ### Installation
    Drag and Drop the resource into your server-files and go to the configuration file located at `/conig.lua`.

    ### Configuration
    Please make sure to configure the script to your need to ensure the best experience. More information can be found [here](./2_Config).
</Steps>
