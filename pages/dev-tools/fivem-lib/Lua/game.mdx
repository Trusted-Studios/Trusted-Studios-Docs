# Game

## AddAdvancedMarker
Adds a marker to the world at the given coords. Can be customized a bit more than `Game.AddMarker`.

```lua
Game.AddAdvancedMarker(id, coords, r, g, b, a, rotation)
```

- id: `number`
- coords: `vec3` | `vec4`
- r: `number`
- g: `number`
- b: `number`
- a: `number`

<Callout type="info">
    This function has to be called every tick.
</Callout>

*example:*
```lua copy
CreateThread(function()
    while true do
        Wait(0)
        Game.AddAdvancedMarker(1, vec3(10.0, 245.0, 130.0), 255, 255, 255, 255, vec3(0.0, 0.0, 180.0))
    end
end)
```

## GetAdvancedSideField
Calculates the side field of an entity with an optional multiplier.

```lua
Game.GetAdvancedSideField(entity, side, multiplier)
```

- entity: `number`
- side: `string<'left' | 'right'>`
- multiplier: `number`

*example:*
```lua copy
local ped = PlayerPedId()
local coords = GetEntityCoords(ped)

local sideField = Game.GetAdvancedSideField(ped, 'left', 2)

print(coords + vec3(sideField.x, sideField.y, 0))
```
