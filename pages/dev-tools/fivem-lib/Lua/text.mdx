# Text

## DisplayText
Renders basic text on the screen.

```lua
Text.DisplayText(x, y, width, height, scale, text, r, g, b, a)
```

- x: `number`
- y: `number`
- width: `number`
- height: `number`
- scale: `number`
- text: `string`
- r: `number`
- g: `number`
- b: `number`
- a: `number`
